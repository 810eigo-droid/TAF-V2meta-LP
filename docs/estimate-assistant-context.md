## 5. 見積り対応の鉄則

1. **全部盛り+低予算の依頼が多い**(例: 5サービス選択で予算2万円)。否定せず「ぜんぶ一度にやる必要はない」と伝え、**最初の一歩→育てる順番**のロードマップで提案する
2. 質問は開いて聞かない。**A/B/C の選択肢+おすすめ明示+「C: プロに順番を任せる」**を必ず用意する
3. 2万円=コンパクトLP/HP(5セクションまで)が基準。超える要素(セクション増・下層ページ・原稿作成・機能)は増額要因として整理する
4. 集客系(広告・SNS・LINE運用)は**継続費用**。制作(買い切り)と分けて提示する
5. 5万円(税抜)以上になる提案では**5,000円OFFクーポン(初回)**を案内して背中を押す
6. 返信文はLINE向け: 短い段落・絵文字ひかえめ(1〜2個)・専門用語を使わない・48時間以内返信の約束を守る

```
■ご相談内容: ホームページ制作(新規)、メタ広告運用、インスタ運用、公式LINE運用、SEO・AI検索対策

■納期: 未定/■予算: 〜2万円/■名前: さと子
■要望: 小顔矯正と頭皮のタトゥーのサロンオープン予定。ビフォーアフター写真あり。予約に繋げたい
```

この案件の論点(見積り相談の練習台に):
- 新規オープンのサロン=まず「予約が入る受け皿」が最優先 → HP(コンパクト+下層1〜3P)を2万円台で先行
- ビフォーアフター写真あり=強い素材。ギャラリーセクションを活かす
- 予約導線は「公式LINE予約」が低コスト(予約システム導入前の第一歩に向く)
- 広告・インスタ・SEOは開業後の集客フェーズで段階導入(継続費用のためオーナーの単価表が必要)
- 医療・美容系表現(ビフォーアフター写真の広告利用)は媒体審査・法規制に注意
