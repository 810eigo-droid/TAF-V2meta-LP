## 2. サービス資料 service-guide.html(最重要ページ・v22)

- 公開: `taf-design.com/lp/service-guide.html`(エックスサーバー `/public_html/lp/` に手動アップロード)
- **Web版は明るいLPトンマナ**(白×ブルー#00A2DE×オレンジ#F6AC0E・広告LPと同系)。紺×ゴールド案は却下済み・暗くしない
- **スマホ文字の確定スケール(オーナー指定)**: 本文18px/備考・カード文16px/見出しはセクション26px・ストーリー28px・表紙30px/クレジット類11px。判定は「タッチ端末(hover:none/pointer:coarse)」+幅759px以下(仮想幅ブラウザ対策)。**PC版は現状で承認済み・変更しない**
- 構成: 表紙(バッジ16個・v20で刷新)→ストーリー(OUR STORY)→目次→私たちについて→サービス(スマホ2列チップ)→なぜ買い切りか→料金→予算から選ぶ→流れ→実績→お客様の声→FAQ+CTA
- ストーリー: 「安く作る。だけじゃない。売れる形で、安く作る。」/「WEB制作の価格が高いのは〜名残です→その見積もり、半分は"いらない費用"かも」の順/費用比較バー(従来30万 vs AI制作22,000+浮いた予算→広告)/STEP1-3/締めの下に0円公開セット注記
- 導線: 右下「📄 PDF版はこちら」(オーナー指定でPDF優先)/左下「☰ 目次」(1画面スクロールで出現)/セクション末尾リンク3つ(料金へ・この品質で見積もってみる・私も頼んでみる)/PDFは目次末尾と最終CTAにも
- 画像(すべて `/lp/images/`): `guide-hero.webp`(表紙4:3)/`guide-story.webp`(ストーリー16:9)/`lp-daihyo.webp`+`lp-daihyo-sp.webp`(代表写真PC/SP切替16:9)。**固定比率ボックス+object-fitで縦横比を強制**・未配置なら自動非表示・webp→jpg→pngフォールバックあり
- 出現アニメ(IntersectionObserver)+保険(2.5秒で全表示)。強制ダークモード無効化(color-scheme:light only)済み

### PDF版の更新手順(重要)
- **A4印刷用の原稿は `service-guide-print.html`**(Web版とは別ファイル。文言修正は両方に入れること)
- PDF生成: Playwright/Chromiumで print原稿 → A4・printBackground・margin0 → `service-guide.pdf`(12ページ・約7.6MB)
- 公開先: `taf-design.com/lp/service-guide.pdf`(資料内のPDFリンクの飛び先)
