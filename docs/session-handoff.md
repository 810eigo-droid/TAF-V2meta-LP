## 3. TAF-V2meta-LP の現状

### wordpress/order-page-block.html(/order/ お見積りページ)
WordPressカスタムHTMLブロックに丸ごと貼る自己完結ブロック。機能:
- ご相談内容15項目(複数選択)。**2つ以上選ぶと「まず最初に始めたいもの」必須質問が動的出現**
- 条件表示: 下層ページ数(HP時)/分量・原稿・作り方・横幅・機能(制作時)/WP組み立て方(WordPress時)/サーバー(サイト系)/色・雰囲気(SEO・AI・その他以外)
- 料金表ボタン(taf-design.com/price/ 別タブ)+費用感確認の案内
- 納期(2週間以内/1ヶ月/未定+日付)、予算、URL×3、名前、メール欄
- 送信: LINE(依頼文コピー→lin.ee/pqfXTUJ)/メール(MAIL_TO=info@taf-design.com 設定済み)
- 依頼文は■形式で自動生成
- このページだけパンくず・ヘッダーの見積りボタンを非表示
- 重要文言「送信=ご契約ではありません。お見積りを見てから、ゆっくりご検討ください。」は**残す方針で確定**
- 「お見積り後のキャンセル可」は削除済み

### wordpress/coupon-page-blocks.html(/coupon/ クーポンページ)
- **全画面オーバーレイ構造**(テーマCSSの影響を受けず内部スクロール)
- ブロック順: BLOCK1(カスタムHTML)→ CF7ショートコード `[contact-form-7 id="ab758b8"]` → BLOCK3(カスタムHTML)
- BLOCK1の閉じタグがBLOCK3にある構造のため、エディタの「無効なHTML」警告は無視してよい
- クーポン条件: **5,000円OFF・5万円以上の注文・初回1回限り・取得から1年有効**
- [ ] 宿題: 5万円の税込/税抜表記の確定

### service-guide.html(サービス資料・リンク共有で配布)
- P10 CTA: LINEメイン(緑ボタン)+サブ3ボックス(ご注文フォーム「お見積り(無料)はこちら」/公式サイト/SMS)
- 文言は「事業主・副業オーナー」(個人事業主から変更済み)
- 見積り返信の約束は**48時間以内**(orderページと統一)

### docs/estimate-assistant-context.md(見積もりアシスタント資料)
- claude.aiに「TAF見積もりアシスタント」プロジェクト**作成済み**(説明欄に全文貼り付け方式)
- 内容: 確定価格表/未確定価格(創作禁止)/フォーム項目/対応の鉄則(A/B/C選択式で優先順位を聞く)/返信テンプレ/実例(さと子様・サロン案件)
- [ ] 宿題: 広告運用月額・セクション追加単価などの未確定価格が決まったら更新→プロジェクトのナレッジ差し替え
