## 5.5 リンク運用ルール(2026-07-19確定・全チャット共通)

- **通常のWebリンクは同一タブで開く(`target="_blank"`禁止)**。スマホユーザーはブラウザの「戻る」でしか帰れないため。別タブは迷子のもと
- 例外: tel: / mailto: / sms: / LINE(lin.ee)などアプリ起動系はそのままでよい
- /order/ フォームは離脱時に入力内容を自動保存・復元するので、同一タブ遷移でも入力は消えない
- リンク先の使い分け: 見積もり=/order/ ・相談・資料=/offer2/ ・**予算から選ぶ=/budget/(公式HP側で新設の予算ハブ)** ・料金表=/price/
- /budget/ への導線設置済み: 資料P6(オレンジボタン+プラン別リンク)/資料P10クロージング/クーポンページ/見積もりアシスタント資料
