MONTHLY CHECKLIST

毎月の作業は、この3つだけ。

月に1回、日を決めて(例: 毎月1日・月初の月曜など)、上から順に進めてください。ぜんぶで15分ほどです。

✓ 月イチ15分チェックリスト
  • 作業1 ｜ バックアップの確認自動バックアップがきちんと動いているかを確認します 約5分
  • 作業2 ｜ 更新(アップデート)WordPress・プラグイン・テーマを最新の状態にします 約5分
  • 作業3 ｜ サイトの健康チェック表示くずれやフォームの故障がないかを確認します 約5分
大事な順番 必ず「作業1(バックアップ確認)→作業2(更新)」の順で行ってください。先にバックアップがあることを確かめてから更新すれば、万一更新で不具合が起きても元に戻せます。
迷ったら 少しでも「これ、押していいのかな?」と迷ったら、無理に進めず作業を止めて大丈夫です。巻末の連絡先からいつでもご相談ください(相談は無料です)。

バックアップの確認

約5分

納品時に設定した「自動バックアップ」が、きちんと動き続けているかを確認します。

  1. 1WordPressの管理画面にログインするログインする場所(URL)は、巻末の「あなたのサイト情報」メモをご覧ください。
  2. 2バックアップの一覧画面を開く納品時にご案内した場所です(メニューの位置は巻末メモに記入してあります)。
  3. 3いちばん新しいバックアップの日付を見る「ここ1週間以内」の日付になっていればOK。自動バックアップは正常に動いています。
  4. 4(おすすめ)最新のバックアップを自分のパソコンにもダウンロードしておくサーバーの外にも控えがあると、さらに安心です。納品時にお渡ししたZIPと同じ場所に保存しておきましょう。
⚠ こんなときはご相談ください
・いちばん新しいバックアップの日付が1ヶ月以上前になっている
・バックアップの一覧画面が開けない・表示がおかしい
→ 自動バックアップが止まっている可能性があります。そのままにせず、巻末の連絡先までお知らせください。

STEP 2
2

更新(アップデート)

約5分

WordPress本体・プラグイン・テーマを最新にします。更新は、サイトを守るいちばんの基本です。

  1. 1作業1のバックアップ確認が済んでいることをもう一度確かめるバックアップがあれば、更新で万一不具合が起きても元に戻せます。
  2. 2管理画面の左メニュー「ダッシュボード → 更新」を開く更新できるものがあると、数字つきのマークが出ています。
  3. 3プラグインを1つずつ更新するまとめて全部ではなく、1つ更新するたびに次の手順4の表示確認をはさむと、原因の切り分けがかんたんになります。
  4. 4更新のたびに、サイトの表示を確認する別のタブでトップページを開いて、見た目が崩れていないかをサッと確認します。
  5. 5最後にテーマ → WordPress本体の順で更新するここでも、更新後にトップページの表示を確認してください。
⚠ こんなときはご相談ください
・更新したらサイトの表示が崩れた・真っ白になった
・「メジャーアップデート」など大きな更新の案内が出ていて、押すのが不安
→ 慌てなくて大丈夫です。バックアップから元に戻せます。操作を止めて、そのままご連絡ください。

STEP 3
3

サイトの健康チェック

約5分

お客様と同じ目線で、サイトがきちんと動いているかを確認します。スマホとパソコンの両方で見るのがポイントです。

  1. 1スマホでトップページを開く文字や画像の崩れ・表示されない画像がないかを、上から下までスクロールして確認します。
  2. 2パソコンでもトップページを開くスマホと同じように、ざっと全体を確認します。
  3. 3お問い合わせフォームからテスト送信する名前に「テスト」と書いて自分で送信し、通知メールが届くかを確認します。フォームの故障は気づきにくく、機会損失に直結します。
  4. 4ページの表示スピードに極端な変化がないか感じてみるいつもよりあきらかに遅い場合は、何か問題が起きているサインのことがあります。
⚠ こんなときはすぐご相談ください
・見覚えのない広告・ページ・英語の文章が表示されている
・「このサイトは安全ではありません」のような警告が出る
・フォームのテスト送信が届かない
→ 放置すると被害が広がることがあります。気づいた時点でご連絡ください。
